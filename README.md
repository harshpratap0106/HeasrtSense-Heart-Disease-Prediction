# вқӨпёҸ HeartSense: Heart Disease Prediction System

An intelligent and interpretable Heart Disease Prediction System that integrates **Machine Learning**, **ANFIS (Adaptive Neuro-Fuzzy Inference System)**, and **IVCFS (Interval-Valued Complex Fuzzy Sets)** to provide accurate, explainable, and uncertainty-aware predictions.

---

## рҹҡҖ Project Overview

HeartSense is designed to assist in early detection of heart disease using clinical data. The system combines high-performance machine learning with interpretability and uncertainty modeling to ensure reliable and transparent predictions.

The model predicts:
- Risk of heart disease
- Confidence level of prediction
- Interval-based uncertainty range
- Risk classification (Low / Moderate / High)

---

## рҹ§  Key Features

- вң… **High Accuracy Prediction** using Gradient Boosting
- вң… **Interpretability** using ANFIS (fuzzy rule-based explanations)
- вң… **Uncertainty Handling** using IVCFS (interval predictions + confidence)
- вң… **Interactive Web App** built with Streamlit
- вң… **Real-Time Predictions** with user input
- вң… **Medical-Oriented Risk Classification**
